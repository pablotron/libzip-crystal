class Zip::StringSource

  • Zip::StringSource
  • Zip::Source
  • Reference
  • Object

Overview

Use string as a source for Archive#add or Archive#replace.

Example

# create a new string source
src = Zip::StringSource.new(zip, "test string")

# add to archive as "foo.txt"
zip.add("foo.txt", src)
